# Examples

Example notebooks using DFIpy can be found at [https://github.com/thegeneralsystem/dfipy-examples](https://github.com/thegeneralsystem/dfipy-examples).  DFIpy can run anywhere there is internet access.  Click the badges below to open the example notebooks in Binder.

- [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/thegeneralsystem/dfipy-examples.git/HEAD) - Examples Repo
  - [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/thegeneralsystem/dfipy-examples.git/HEAD?labpath=examples%2Fdfipy_quick_start_geolife.ipynb) - Example notebook showing basic DFI queries against the [Geolife dataset](https://www.microsoft.com/en-us/research/publication/geolife-gps-trajectory-dataset-user-guide/).
  - [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/thegeneralsystem/dfipy-examples.git/HEAD?labpath=examples%2Fdfipy_quick_start_syn_traffic.ipynb) - An analysis of activity in the London Congestion Zone using a synthetic 92 billion record dataset.
